# ud-react-template

## Getting Started

#### Clone the repo
```bash
git clone https://github.com/utilidev/ud-react-template.git
```

#### Install the dependancies
```bash
npm install
```
#### Start the dev server
```bash
npm run dev-server
```
### OR

#### Run tests
```bash
npm test
```

## Build for Production
```bash
npm run build
```
