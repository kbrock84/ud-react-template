# ud-react-template

<br />

## Getting Started

#### Clone the repo

```bash
git clone https://github.com/utilidev/ud-react-template.git
```

#### Install the dependancies

```bash
npm install
```

#### Start the dev server

```bash
npm run dev-server
```

#### Run tests

```bash
npm test
```

<br />

## Build for Production

```bash
npm run build
```
